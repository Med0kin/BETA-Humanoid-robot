## Humanoid robot repository

BETA is project that aims to create humanoid robot that's capable of walking, seeing and talking, just like regular human!

Along with idea, project started back in 2018, changing it's form with new upgrades. Currently it's developed as project in students scientific association "Ko≈Ço Naukowe Humanoid" where we're regulary posting his new features.
